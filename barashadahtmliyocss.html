Barashada HTML iyo CSS: Hagaha Bilowga

Barashada HTML iyo CSS waa talaabada ugu muhiimsan ee qof kasta oo raba inuu website dhiso. HTML (HyperText Markup Language) wuxuu ka masuul yahay qaab-dhismeedka webseed-kaaga, halka CSS (Cascading Style Sheets) uu ka mas'uul yahay muuqaalka, midabka, iyo layout-ka. Fahamka labadan luqad waa aasaaska web development waana meesha ama albaabka saxda ah ee uu qofka ka soo galayo xirfada web-develper.

1. Waa Maxay HTML?

HTML waa luqad lagu dhiso qaab-dhismeedka website-ka. Waxay ka kooban tahay elements iyo tags kuwaas oo qeexaya qaybaha bogga sida headings, paragraphs, images, links, lists, iyo forms. sida aan horay sawirka kuugu soo bandhignay, HTML qaab noocaas ayaa loo qorayaa lakiin si fudud aan usii kala dhigdhigno. Tusaale: 

<h1>Hello World!</h1>
<p>Tani waa paragraph HTML ah.</p>

2. Waa Maxay CSS?

CSS waa luqadda mas'uulka ka ah muuqaalka webseed-ka. Waxaad ku hagaajin kartaa midabada, fonts, spacing, layout, hover effects, iyo animations. CSS waxay ka dhigtaa website-ka mid soo jiidasho leh oo professional ah.si lamid ah sawirka kore, CSS waxaynu u qori karnaa qaab noocaan bal aynu faahfaahino tusaalaha css. 

p {
  color: #1e293b;
  font-size: 16px;
  line-height: 1.6;
}

3. Qaab-dhismeedka Aasaasiga ah ee HTML + CSS

Marka la isku daro HTML iyo CSS, waxaad dhisi kartaa bog caadi ah oo nadiif ah. Tusaale layout fudud: 

<!DOCTYPE html>
<html lang="so">
<head>
  <meta charset="UTF-8">
  <title>Website Example</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Hello HTML & CSS!</h1>
  <p>Boggan wuxuu tusaale u yahay barashada web development.</p>
</body>
</html>
